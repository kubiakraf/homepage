Polecenia Terminala

Konsola linux

  • ls - lista plików
  • pwd - wyświetla informację w którym katalogu własnie się znajdujemy
  • cd - change directory - wejście do katalogu (tab dopełnia nazwę)
  • mkdir - make directory - stwórz katalog
  • cd .. - wyjście z katalogu powrót do katalogu wyżej
  • cd dev/homepage/ - wejście bezpośrednio do katalgu homepage
  • code . - uruchamia VSC z dostępem do plików z katalogu w któym został otwarty
  • cd ~ - szybki powrót do katalogu domowego
  • ifconfig - pokazuje informacje o karcie sieciowej i sieci

Konsola MacOS

  • command + space: terminal
  • ~ (tylda) oznacza, że jesteśmy w katalogu domowym
  • ls - lista katalogów i plików
  • pwd - wyświetla informację w którym katalogu własnie się znajdujemy
  • cd - change directory - wejście do katalogu (tab dopełnia nazwę)
  • mkdir - make directory - stwórz katalog
  • cd .. - wyjście z katalogu powrót do katalogu wyżej
  • cd dev/homepage/ - wejście bezpośrednio do katalgu homepage
  • code . - uruchamia VSC z dostępem do plików z katalogu w któym został otwarty
  • VSC: Shift + Cmd + P : shell -> Install 'code' command in PATH - doinstalowuje uruchamianie VSC z terminala poleceniem code . - jeśli nie zadziałało.
  • cd ~ - szybki powrót do katalogu domowego
  • ifconfig - pokazuje informacje o karcie sieciowej i sieci

Konsola na Windowsie:

  • cd C:\ - przenosi do katalogu głównego z każdego miejsca
  • podwójny TAB przełącza pomiędzy propozycjami terminala w dokończeniu wpisywanej nazwy
  • jeśli nazwa katalogu, pliku ma w sobie spacje należy zamykać całą w cudzysłów
  • stawiając \ po nazwie katalogu terminal podpowie katalogi znajdujące się w środku
  • cd - bez żadnych parametrów wyświetla ścieżkę w jakiej aktualnie się znajdujemy
  • dir - pokazuje katalogi i pliki w tym katalogu
  • mkdir - tworzenie katalogów
  • code . - uruchamia VSC z dostępem do plików z katalogu w któym został otwarty
  • VSC: Shift + Cmd + P : shell -> Install 'code' command in PATH - doinstalowuje uruchamianie VSC z terminala poleceniem code . - jeśli nie zadziałało.
  • cd .. - wyjście z katalogu powrót do katalogu wyżej
  • ipconfig - wyświetla ustawienia sieci IPv4 Address - do łączenia się z serwerem lokalnym
