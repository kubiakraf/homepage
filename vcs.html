Najważniejsze ustawienia

  • Ctrl + Shift + P - otwiera okno wpisywania poleceń do edytora
  • > Format document - formatuje kod aby był bardziej przejrzysty.
  • > Indent Using Tabs - wcięcia zmienić na 2.

Rozszerzenia

  • live server (lub nowszy five server) - tworzy lokalny serwer
    • http://127.0.0.1:5500/katalog/index.html
    • "127.0.0.1" - podmienić IP Adresem lokalnym
    • Można na innym urządzeniu, w tej samej sieci WiFi wyświetlić stronę podając cały adres
    • http://192.168.1.95:5500/katalog/index.html
  • material icon theme
  • htmltagwrap - nie dodaje znacznika zamykającego
  • Prettier - Code formatter
    • Ctrl Shift P - stworzenie pliku konfiguracyjnego
    • Prettier: Create Configuration File
    • Wybierz folder i ok
    • można zmienić tylko domyślny tabWidth z 2 na 4
    • Ctrl Shift P : Preferences: Open User Settings
    • Format on Save : on
    • Formatter - Editor: Default Formatter: Prettier - Code Formatter
  • auto rename tag
  • svg viewer

Przydatne skróty

  • Alt + Shift + ↓ - kopiuje tą jedną linię
  • Alt Z - zawija tekst w VCS ale tylko tymczasowo do następnego uruchomienia.
  • Lorem5 [tab] - 100 słów
  • Lorem*5 [tab] - x 5
  • Shift Ctrl F - format document - przyjazny dla oka widok
  • Setting / Format On Save - robi to przy każdym save
  • Ctrl [/?] - zaznaczony obszar kodu zamienia w komentarz.
    Są ważne dla innych programistów ale też semantyka opisowa dla <div>, <section>, <article>.
  • Ctrl ~ otwiera terminal
  • Ctrl + B - ukrywa sidebar z „eksploratorem”
  • Ctrl + D - zaznacza cały wyraz zaznacza w tekście, następne ponieżej i umożliwia edytowanie na raz paru np. tagów
  • Ctrl + D x2... - kolejne użycie zaznacza w tekście, następne ponieżej i umożliwia edytowanie na raz paru np. tagów
  • Ctrl + Alt + [→] - otwartą zakładkę przenosi do nowego okna po prawej
  • Ctrl + Alt + [←] - zamyka prawy podział ekranu
  • Ctrl + Alt + [↑]/[↓] - duplikuj "karetkę" aby móc jednocześnie w kilku miejscach wprowadzać zmiany
  • . (kropka) i [Tab] = Emet tworzy: <div class=""> </div>
  • .nazwa i [Tab] = Emet tworzy: <div class="nazwa"> </div>
  • Alt i strzałki góra/dół - przesuwają tekst linii góra/dół
